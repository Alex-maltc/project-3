# Election Scraper 2017

Tento projekt slouží k automatickému vytěžování dat z výsledků voleb do Poslanecké sněmovny v roce 2017 z webu [volby.cz](https://www.volby.cz).

## Instalace

Pro spuštění projektu je nutné mít nainstalovaný Python 3. Následně si nainstalujte potřebné knihovny pomocí přiloženého souboru `requirements.txt`:

pip install -r requirements.txt
